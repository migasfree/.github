<h1 align="center">Migasfree</h1>
Migasfree is free software designed to manage dynamic software repositories based on attributes and schedules. Its main functionality is to offer package repositories that are available to clients based on their attributes.
To learn more, [check the documentation](http://fun-with-migasfree.readthedocs.org/).
